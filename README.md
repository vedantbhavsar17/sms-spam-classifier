# 📩 Spam SMS Classifier

A Machine Learning web app built using Streamlit that classifies SMS messages as **Spam** or **Not Spam**.

---

## 🚀 Features

- Text preprocessing using NLTK
- TF-IDF Vectorization
- Naive Bayes Model
- Interactive Streamlit UI
- Real-time predictions

---

## 🧠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLTK

---

## 📦 Installation

```bash
git clone https://github.com/your-username/sms-spam-classifier.git
cd sms-spam-classifier
pip install -r requirements.txt
```

## ▶️ Run App

```bash
streamlit run app.py
```

## 📊 Example

**Input:**

`Win cash prizes up to $5000. Limited offer!`

**Output:**

`Spam 🚨`
